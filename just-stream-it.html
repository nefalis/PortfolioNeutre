Just Stream It

Application web qui répertorie et classe les films

Contexte du projet :
Création d'une application web afin d'aider une association a monter en croissance.

Objectif :
Interface utilisateur pour une application web qui répertorie et classe les films en fonction de leur note. L'interface est composé de 5 parties : Meilleur film - Les films les mieux notés - Catégorie 1 - Catégorie 2 - Catégorie libre avec un menu déroulant pour choisir une catégorie. La page se connecte à une API afin de récupérer les différentes informations.

HTML - CSS - JavaScript
